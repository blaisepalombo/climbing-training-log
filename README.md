# climbing-training-log

Climbing Training Log is a web application designed to help climbers track their training sessions and monitor their progress over time. The app allows users to log workouts, record exercises, sets, reps, and notes, and review their training history. By organizing workout data and displaying progress metrics, the application helps climbers stay consistent with their training and better understand their performance improvements.
